Common Questions
What hardware do I need?
A Mac with Apple Silicon (M1/M2/M3/M4) is ideal. We can also set up on a Linux server or cloud VM if you prefer.
Is my data really private?
Yes. The AI runs entirely on your machine. Your conversations, documents, and workflows never leave your hardware unless you explicitly share them.
What can it actually do?
Email drafting, calendar management, research, reminders, file organization, web browsing, data entry, and hundreds of automations. If it can be done on a computer, it can probably be automated.
How is this different from ChatGPT/Claude?
Those are chat interfaces. This is a full assistant with persistent memory, real integrations, and the ability to take action — not just answer questions.
