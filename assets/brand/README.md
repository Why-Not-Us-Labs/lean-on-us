## Usage

- **Web/App**: Use `logo.svg` for crisp scaling at any size
- **Social**: Use `logo-square.png` or `favicon-512.png`
- **Dark mode**: Use `logo-white.png` or `logo-dark-bg.png`
- **Print**: Use `logo.svg` or `logo-flat-trimmed.png`
